Para executar nosso programa, usando o terminal do linux ou powershell do windons navegue até o diretório 
do projeto 'APS_BD2', depois vá ate a pasta 'dist' onde se encontra o arquivo APS_BD2.jar, em seguida execute a seguinte linha:

java -jar APS_BD2.jar

O programa executará ate encontrar o ultimo indice do banco de dados, caso queira interrompe -lo, basta pressionar
as seguintes teclas Ctrl + c.

O programa executará uma busca no banco de dados  a cada 8 segundos, e serializara  as informações obtidas e salvara em
outra tabela chamada 'saida'.

Caso encontre o ultimo registro registro do banco, o sistema será encerrado.
